## Overview

This project replicates the home page of Netflix India and includes the following:

* The main **NETFLIX** effect in the first part of the page
* inputs for entering email ID
* sign-in, language, and Get started buttons
* sections containing videos playing inside a device (videos aligned inside the device frame images)
* Frequently Asked Questions
* footer
<hr>
