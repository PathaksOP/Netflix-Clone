# Netflix Landing Page Clone

A static front-end recreation of the Netflix India landing page, built using HTML and CSS. The project focuses on layout accuracy, visual hierarchy, and responsive design, closely mirroring the structure and feel of the original interface.

## Live Demo

View the project here:
🔗 [Open Live Website](https://pathaksop.github.io/Netflix-Clone/)

## Overview

This project replicates the core sections of Netflix’s homepage, including:

* Hero section with call-to-action
* Feature highlights (TV, mobile, multi-device streaming, kids profiles)
* FAQ layout
* Footer with structured links

The emphasis is on clean styling, positioning, and responsive behavior rather than functionality or backend integration.

## Tech Stack

* HTML5
* CSS3

## Features

* Structured and semantic HTML layout
* Responsive design for different screen sizes
* Layered hero section with overlay effect
* Embedded video elements for visual realism
* Reusable utility classes for buttons and layout
* FAQ section styled for interaction (hover effects)

## How to Run

1. Clone or download the repository
2. Open `index.html` in any modern browser

No build tools or dependencies are required.

## Notes

* This is a purely front-end project intended for learning and practice
* No JavaScript functionality (e.g., FAQ toggle, form handling) is implemented
* Some assets are externally sourced and may depend on network availability
